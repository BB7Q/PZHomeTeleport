title=HomeTeleport
description=[h1]HomeTeleport - 回家传送[/h1]
description=[h2]背景介绍[/h2]
description=在体验房车模组的过程中，我常常发现对当前使用的车辆或房间布局不够满意。无论是车辆的外观、内部空间，还是建筑的布局和位置，都可能无法完全符合个人偏好。这种限制让我萌生了开发一个更灵活传送系统的想法——让玩家能够自由选择心仪的车辆和建筑作为传送目标，真正实现"随心所欲"的生存体验。
description=[h2]使用方法[/h2]
description=[list]
description=    [*] 设置家的位置：在地面上右键点击，选择"设置家的位置"
description=    [*] 传送回家：在车辆上按V键，选择回家图标
description=    [*] 返回车辆：从车辆传送回家后，在世界界面右键点击选择"返回车辆"
description=[/list]
description=[h2]功能特点[/h2]
description=[list]
description=    [*] 车辆保留：从车辆传送回家后会记录车辆位置
description=    [*] 沙箱支持：可配置固定家的位置
description=    [*] 沙箱支持限制模式和非限制模式：限制模式只可绑定一辆车（可随时更换），非限制模式任意车辆都可以传送
description=[/list]
description=[h2]沙箱配置[/h2]
description=在游戏沙箱设置中可配置固定家的位置：
description=[list]
description=    [*] HomeTeleport.EnableFixedHomePosition - 启用固定家的位置
description=    [*] HomeTeleport.FixedHomePositionX - 固定家的X坐标
description=    [*] HomeTeleport.FixedHomePositionY - 固定家的Y坐标
description=    [*] HomeTeleport.FixedHomePositionZ - 固定家的Z坐标
description=[/list]
description=启用固定位置后，玩家无法手动修改家的位置，系统将使用预设坐标。
description=[h2]注意事项[/h2]
description=[list]
description=    [*] 传送回家功能只能在车辆上使用
description=    [*] 设置家的位置需要在地面上
description=    [*] 多人模式尚未完成
description=[/list]
description=[h2]待办事项[/h2]
description=[list]
description=    [*] 支持多人模式
description=[/list]
description=[hr]
description=[h2]Background[/h2]
description=During my experience with RV mods, I frequently encountered situations where the available vehicles or room layouts didn't meet my expectations. Whether it was the vehicle's appearance, interior space, or the building's design and location, there was always room for improvement. This limitation inspired me to develop a more flexible teleportation system - one that empowers players to freely select their preferred vehicles and buildings as teleportation destinations, truly enabling a "do whatever you want" survival experience.
description=[h2]How to Use[/h2]
description=[list]
description=    [*] Set Home Location: Right-click on the ground and select "Set Home Location"
description=    [*] Teleport Home: Press V key in vehicles and select the home icon
description=    [*] Return to Vehicle: After teleporting from a vehicle, right-click in the world and select "Return to Vehicle"
description=[/list]
description=[h2]Features[/h2]
description=[list]
description=    [*] Vehicle Preservation: Records vehicle position when teleporting from vehicles
description=    [*] Sandbox Support: Configurable fixed home position
description=    [*] Sandbox supports restricted and unrestricted modes: restricted mode allows binding only one vehicle (can be changed anytime), unrestricted mode allows teleporting from any vehicle
description=[/list]
description=[h2]Sandbox Configuration[/h2]
description=Configure fixed home position in game sandbox settings:
description=[list]
description=    [*] HomeTeleport.EnableFixedHomePosition - Enable fixed home position
description=    [*] HomeTeleport.FixedHomePositionX - Fixed home X coordinate
description=    [*] HomeTeleport.FixedHomePositionY - Fixed home Y coordinate
description=    [*] HomeTeleport.FixedHomePositionZ - Fixed home Z coordinate
description=[/list]
description=When fixed position is enabled, players cannot manually modify home location and the system will use preset coordinates.
description=[h2]Important Notes[/h2]
description=[list]
description=    [*] Teleport home function can only be used from vehicles
description=    [*] Setting home location requires being on the ground
description=    [*] Multiplayer mode not completed yet
description=[/list]
