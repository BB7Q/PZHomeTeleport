# HomeTeleport

为《僵尸毁灭工程》添加回家功能的模组。

## 功能

- **设置家**：在车外通过右键菜单"设置家"将当前位置设置为家
- **回家**：在车内通过车辆轮盘菜单"回家"传送回设置的家位置
- **返回车辆**：从车辆回家后，可以通过右键菜单"返回车辆"回到之前的位置

## 使用说明

### 设置家

1. 在车外（不在车辆中）
2. 右键打开菜单
3. 选择"设置家"选项
4. 当你当前位置已成功设置为家

### 回家

1. 进入车辆
2. 打开车辆轮盘菜单
3. 选择"回家"选项
4. 传送到设置的家位置

### 返回车辆

如果你是从车辆回家的，可以通过右键菜单"返回车辆"选项回到原来的车辆位置。

## 注意事项

- 回家功能只有在车内才能使用
- 回家前会检查周围是否有僵尸，有僵尸在附近时无法传送
- 家的位置会自动保存，重新登录后仍然有效
- 支持多人游戏，每个玩家可以设置自己的家位置

## 兼容性

- 适用于 Project Zomboid B42 版本
- 支持单人和多人模式
